Selector[atributo]

Los enlaces con atributo target tendrán fondo amarillo

google.com marca.com wikipedia.org

Selector[atributo="valor"]

Los enlaces con el atributo target="_blank" tendrán el fondo amarillo

google.com marca.com wikipedia.org

Selector[atributo~="valor"]

Todas las imágenes con la palabra "flor" en el atributo title tendrán un borde amarillo.

Selector[atributo|="valor"]

Todo el texto que pertenezca a una clase que empiece por la palabra top (palabra completa) tendrá fondo amarillo.

Selector[atributo^="valor"]

Todo el texto que pertenezca a una clase que empiece por top tendrá fondo amarillo.

Bienvenidos

Selector[atributo$="valor"]

Todo el texto que pertenezca a una clase que termine por test tendrá fondo amarillo.

El primer elemento div class="first_test".
El segundo elemento div class="second".
El tercer elemento div class="my-test".

Esto es un párrafo cualquiera class="mytest".

Selector[atributo*="valor"]

Todo el texto que pertenezca a una clase que contenga por "te" tendrá fondo amarillo.

El primer elemento div class="first_test".
El segundo elemento div class="second".
El tercer elemento div class="my-test".

Esto es un párrafo cualquiera class="mytest".

Los selectores de atributos pueden ser útiles para diseñar formularios sin clase o ID
