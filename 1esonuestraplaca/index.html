Diccionario Lectura Facilitada

Autómata

Definición:

Máquina capaz de funcionar sin ayuda humana.

Ejemplo:

El autómata hizo la tarea sin ayuda humana.

Circuitos

Definición:

Plural de circuito. Conjunto de componentes eléctricos que hacen una función.

Ejemplo:

El circuito hace que la bombilla se encienda con una pila.

Código abierto

Definición:

Es un código que se puede usar y estudiar porque está publicado.

Ejemplo:

El programa Scratch es de código abierto.

Entorno de programación

Definición:

Es un programa que permite escribir código y transformarlo en una aplicación informática.

Ejemplo:

Python es un entorno de programación.

Entrada

Definición:

Elemento que lleva información a un sistema informático.

Ejemplo:

El ratón y el teclado son entradas de un ordenador.

Frankenstein

Definición:

Novela en la que crean a un hombre.

Ejemplo:

Frankenstein es una mezcla entre un monstruo y un humano.

Laboratorio

Definición:

Lugar donde se investiga y hacen experimentos.

Ejemplo:

En el laboratorio hacen experimentos.

Programar

Definición:

Dar instrucciones a una máquina o aparato para que realice funciones.

Ejemplo:

Vamos a programar el robot.

LED

Definición:

Es una fuente de luz de bajo consumo y que dura mucho sin estropearse.

Ejemplo:

El teléfono móvil lleva una luz LED para iluminar.

Mary Shelley

Definición:

Mujer que escribio el libro de Frankenstein.

Ejemplo:

Mary Shelley escribió Frankenstein.

Microcontrolador

Definición:

Un microcontrolador es un circuito integrado programable, capaz de ejecutar las órdenes grabadas en su memoria.

Ejemplo:

La placa Microbit lleva un microcontrolador para controlar sus funciones.

RAM

Definición:

Es la memoria principal de un dispositivo.

Ejemplo:

La memoria RAM almacena la información de los sensores.

Robótica

Definición:

Es la ciencia que se ocupa del diseño y construcción de los robots.

Ejemplo:

La robótica está evolucionando mucho en los últimos años.

Salida

Definición:

Elemento que muestra la información en un sistema informático.

Ejemplo:

La pantalla es un componente de salida en un ordenador.

Silicio

Definición:

Es un material que sirve para conducir o aislar la corriente eléctrica. Se utiliza en la industria electrónica y microelectrónica, para fabricar componentes.

Ejemplo:

Muchas piezas electrónicas están hechas de silicio.
